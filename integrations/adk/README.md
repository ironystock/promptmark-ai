# ADK Integration

Reference integration connecting Promptmark with the Google Agent Development Kit (ADK).

## Status

Planned — not yet implemented.

## Overview

This integration will demonstrate how to use Promptmark as a prompt backend for ADK agents, leveraging MCP tools for prompt retrieval, versioning, and composition within ADK workflows.

## MCP Tools

Key Promptmark MCP tools for this integration:
- `list_prompts` — browse available prompts
- `get_prompt` — retrieve prompt content with template variables resolved
- `search_prompts` — find prompts by content or metadata
- `list_tags` — discover prompt categories
