# OpenClaw Integration

Reference integration connecting Promptmark with the OpenClaw agent framework.

## Status

Planned — not yet implemented.

## Overview

This integration will demonstrate how to use Promptmark's MCP tools from within OpenClaw agent workflows, enabling agents to manage, retrieve, and compose prompts as part of their execution.

## MCP Tools

Key Promptmark MCP tools for this integration:
- `list_prompts` — browse available prompts
- `get_prompt` — retrieve prompt content with template variables resolved
- `search_prompts` — find prompts by content or metadata
- `create_prompt` / `update_prompt` — manage prompts programmatically
