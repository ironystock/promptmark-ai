# OpenAI Agents SDK Integration

Reference integration connecting Promptmark with the OpenAI Agents SDK.

## Status

Planned — not yet implemented.

## Overview

This integration will demonstrate how to use Promptmark's MCP server as a tool provider for OpenAI Agents, enabling agents to manage and retrieve prompts through the Agents SDK's MCP support.

## MCP Tools

Key Promptmark MCP tools for this integration:
- `list_prompts` — browse available prompts
- `get_prompt` — retrieve prompt content with template variables resolved
- `search_prompts` — find prompts by content or metadata
- `create_prompt` / `update_prompt` — manage prompts programmatically
