# TUI Client

A terminal UI client for Promptmark, providing a rich interactive experience for prompt management from the terminal.

## Status

Planned — not yet implemented.

## Vision

A bubbletea-based TUI offering:
- Browse and search prompts
- Create and edit prompts with template variable support
- Version history navigation
- Collection management
- MCP server integration for AI-assisted workflows

## Tech Stack (Planned)

- [Bubble Tea](https://github.com/charmbracelet/bubbletea) — TUI framework
- [Lip Gloss](https://github.com/charmbracelet/lipgloss) — Styling
- Promptmark REST API / MCP for backend communication
