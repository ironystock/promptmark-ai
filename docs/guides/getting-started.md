# Getting Started

Get up and running with Promptmark in minutes.

## Sign Up

1. Visit [promptmark.ai](https://promptmark.ai)
2. Click "Sign In" to authenticate via Civic Auth
3. You'll get your own isolated database — your prompts are yours alone

## Create Your First Prompt

1. Click "New Prompt" from the dashboard
2. Choose a creation mode:
   - **Blank** — start from scratch
   - **Guided** — step-by-step wizard
   - **AI Consultation** — let AI help you craft the prompt
3. Add a title, content, tags, and optional template variables
4. Save — your prompt is versioned automatically

## Template Variables

Promptmark supports typed template variables in your prompts:

```
Write a {{style}} blog post about {{topic}} for {{audience}}.
```

Variables can have types (text, number, select), default values, and descriptions.

## Connect MCP

Use Promptmark with AI agents via the Model Context Protocol. See the [MCP Setup Guide](mcp-setup.md) for detailed instructions.
